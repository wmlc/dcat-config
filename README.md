# Dcat Admin Extension


### composer 安装

> composer require wmlc/dcat-config

### dcat config 配置

> admin_setting 添加 wmlc::admin_config 字段

> key=>value形式保存

> 可以替换系统config 或env 配置
> 使用 config("key");
